Soccer Widgets: use shortcodes to deliver updated soccer data like various table rankings and football results by competition.

== Description ==

Show football scores and table rankings in your WordPress website using Soccer Widgets shortcodes. Free data, updated automatically, from 250+ football competitions from all over the world, available in multiple languages.

Features:

- widgets with football table rankings and latest match results from 250+ competitions around the world
- data available for pull in 16 languages (English, Spanish, French, German, Swedish, Danish, Norwegian, Finnish, Dutch, Italian, Portuguese, Polish, Romanian, Hungarian, Czech, Greek)

How to use:

- install the plugin and activate it in your Wordpress website
- in the left side menu in Admin area look for Soccer Widgets item
- go to Soccer Widgets to generate the shortcode for any data options like results and table rankings
- copy the shortcode and paste it in your blog post or page
- optional: most data in widgets are presented in table formats, with 100% width. You can use html code to control the size of the width
- optional: data tables from these widgets make use of your website CSS style. However if you do not like how they look in your pages, play with the Customization option of the plugin to change style colors

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/webeki-soccer-scores` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Use the WP Admin->Soccer Stats screen to generate widget shortcodes
4. Copy the shortcode and implement it in your post or page where you want the widget to appear
5. Optional: Use the WP Admin->Soccer Stats to customize the widget colors with the quick color picker tool


== Frequently Asked Questions ==

= Do I need to update the shortcode regularly to get latest rankings or results? =

No. The shortcode pull the latest resuls and rankings for you. Once implemented, you do not need to update the shortcode again.

= Is it free to use =

Yes, data is provided free for use by SoccerStats247.com. Data should be used as is - you are not permitted to alter the data provided by SoccerStats247.com in any way.
